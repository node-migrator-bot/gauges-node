Class Detail
Fetcher(port, host)
A Fetcher is an object that listens the redis queue and blocks and makes the request and puts back the request into the queue. JSON is used for the transport protocol. Emits events that are status codes, like "200", "404", "302" etc. 
Fetcher = require('superrequest').Fetcher;
fetcher = new Fetcher(6379, 'my.hostname.com');

fetcher.on('200', function(res) {
  // Do something.
});

port
host
